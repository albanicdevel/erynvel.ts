U **can** use on `ts`:

```ts
import {
    Client,
    Message,
    Events
} from "@albanicdevel/erynvell"

const client = new Client({
    token: ""
});

client.login();
```

And on `js`:
```js

const {
    Client,
    Message,
    Events
} = require("@albanicdevel/erynvell");

const client = new Client({
    token: ""
});

client.login();
```
